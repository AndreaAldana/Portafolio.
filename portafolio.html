Rick And Morty Fansite

ReactJS / Hooks / Bootstrap / Html / Css / JS / Axios

Fecha: 26/11/2020

Puedes verlo en: AQUÍ

Resultado de una prueba técnica en la cual me pidieron ciertos requisitos. Posee carruseles con datos de Rick And Morty los cuales pueden ser vistos a detalle, y puede filtrarse por categoría. Consume dos API, una para simular un login y registro, usando también AuthGuard para autenticación del usuario, logueo consistente y protección de rutas.
